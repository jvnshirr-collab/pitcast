# VR — Corroded-pipe remaining strength (`b31g.js`)

**Intended use**: screening remaining strength and safe operating pressure of
**blunt metal-loss (corrosion)** on pipelines. **Not** crack-like flaws, SCC, or
sharp gouges (those need API 579 Part 9, outside PitCast's corrosion scope).

**Governing models**
- ASME B31G-2012 **original** (0.667·dL parabolic area).
- **Modified B31G / RSTRENG** basis (0.85·dL effective area).
- **Folias bulging factor** M, short- vs long-defect regimes.

**Cited basis**: ASME B31G-2012; Kiefner & Vieth 1989 (PR-3-805, the Modified-B31G
/ RSTRENG origin); Folias 1965 (*Int. J. Fract.*).

**Verification (worked example)** — `b31g.js:190-198` reproduces **ASME B31G-2012
Appendix B, Example 1**: P_safe = 75.5/1.39 ≈ **54.3 bar (~787 psi)**, matching the
published example. Intact-pipe limit reduces to the **Barlow thin-shell** equation
(closed-form code-verification, SVVP §3.1).

**Validation**: this is a closed-form reproduction of a consensus code, so
validation is **exact** (to rounding), not statistical — the standard *is* the
oracle. Run `node b31g.js` for the embedded worked-case assertion.

**Known limitations**
- Volumetric blunt metal loss only; B31G is explicitly **not** valid for crack-like
  defects, environmentally-assisted cracking, or sharp gouges.
- Single-defect; multi-defect interaction is handled separately by `interaction.js`
  (DNV-RP-F101 / POF-100 / B31G-2012 §3.4.5).
- Flow-stress and Folias forms follow B31G-2012; later RSTRENG effective-area
  refinements (full river-bottom profile) are not implemented.

**Acceptance band**: exact reproduction of the standard's equations (±rounding).
**Reproduce**: `node b31g.js`.
