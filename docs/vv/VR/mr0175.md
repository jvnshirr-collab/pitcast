# VR — Sour-service materials acceptability (`mr0175.js`)

**Intended use**: screening whether a CRA or carbon/low-alloy steel is acceptable
for sour service per ISO 15156-3 / NACE MR0175, returning `IN_SCOPE`, the routing
(Annex A sub-clause or ISO 15156-2 Region), and the governing envelope. **Not** a
substitute for the standard itself or for qualification testing.

**Governing logic**
- Family classification (UNS lookup + Schaeffler/WRC-1992 fallback).
- Annex A "use-without-further-testing" envelope lookup for CRAs.
- ISO 15156-2 **Figure 1 Region** (0/1/2/3) from pH₂S × in-situ pH for
  carbon/low-alloy steel (`_regionLookup`, mr0175.js:119).
- Manufacturing gates: HRC, cold-work, PREN.

**Cited basis**: ANSI/NACE MR0175-2021 / ISO 15156:2020 Parts 1+2+3 + Technical
Circulars 1 & 2.

**Verification (decision tree, `_runTests`, mr0175.js:363+)** — run `node mr0175.js`:
- **Test 1** — 13Cr OCTG @ 120 °C, 8 kPa H₂S → route **A.19**, `IN_SCOPE = true`. ✓
- **Test 2** — 2205 @ 30 kPa H₂S exceeds the 22Cr (A.24) envelope (≤10 kPa) →
  `IN_SCOPE = false`. ✓ (envelope-rejection logic works)
- **Test 5** — carbon steel routed via the Region lookup. ✓

**Validation**: this is a **spec/decision** tool, so validation = **traceability**
of every verdict to the cited ISO 15156-3 table / ISO 15156-2 figure, not a numeric
error metric. Each verdict carries its citation string.

**HONEST LIMITATION (integrity stance, not a defect)**: 18 of 41 Annex A
sub-envelopes are deliberately flagged `needs_review` with **null thresholds** —
they are **not fabricated** from memory. The engine surfaces this flag; a
`needs_review` verdict is **not** a qualified result and must be checked against the
hard-copy standard. Filling these requires the standard's text (see PLAN-tier3 G7).

**Acceptance**: correct routing + envelope traceability; **zero fabricated
thresholds**. **Reproduce**: `node mr0175.js` (`_runTests`).
